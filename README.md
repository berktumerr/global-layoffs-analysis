# Global Layoffs Analysis

## Project Objective

Analyze global layoffs between 2020 and 2026 to identify trends across companies, countries, industries, and time periods.

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed

- Data Understanding
- Data Cleaning
- Company Analysis
- Country Analysis
- Industry Analysis
- Time Series Analysis
- Data Visualization

## Key Findings

- Amazon had the highest number of layoffs.
- The United States dominated global layoffs.
- Retail, Hardware, and Consumer industries were heavily affected.
- Layoffs peaked around late 2022 and early 2023.
